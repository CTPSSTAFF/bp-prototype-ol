Bicycle/Pedestrian Traffic Count Metadata

The following table defines the meaning of the columns in the downloadable bicycle/pedestrian count data CSV file:

Column name Contents
id Unique identification number for a specific count (assigned by MPO)
bp_loc_id Identification number of the location at which a count was taken (assigned by MPO)
count_id Identification number of all the counts collected at one location during the same time period (assigned by MPO)
town The municipality (city or town) in which the counts were collected
description Any details about conditions, observations, or about the count itself
temperature Temperature, measured in degrees Fahrenheit
sky 1 = Sunny; 2 = Partly Cloudy; 3 = Overcast; 4 = Precipitation; -99 = Nothing Recorded
count_type C = Child in carrier; B = Person bicycling (count 2 if in tandem); J = Person jogging; S = Person skating; P = Person walking; W = Person using wheelchair; O = Others
from_st_name Name of the street from which the counted group approached the screen line
from_st_dir Direction from which the counted group approached the screen line: NB = Northbound; EB = Eastbound; SB = Southbound; WB = Westbound; All = All directions recorded together
to_st_name Name of the street toward which the counted group traveled after passing the screen line
to_st_dir Direction toward which the counted group traveled after passing the screen line: NB = Northbound; EB = Eastbound; SB = Southbound; WB = Westbound; All = All directions recorded together
count_date Date of the count (Format: Month/Day/Year)
count_dow Day of the week on which the count was collected: Mon = Monday; Tue = Tuesday; Wed = Wednesday; Thu = Thursday; Fri = Friday; Sat = Saturday; Sun = Sunday
